# Catalog Scraper Mini Project - UCP

## Project Purpose
A modular Python web scraper built for the University of Central Punjab (Tools & Tech for DS). It extracts product data from a static e-commerce test site, navigating through categories, subcategories, and paginated listings.

## Tools Used
- **Language:** Python
- **Package Manager:** `uv` (Required)
- **Libraries:** `BeautifulSoup4`, `Requests`, `Pandas`
- **Version Control:** Git (Feature/Fix Branching Workflow)
